# MINI PROYECTO NRO 1
## Autor
 **Pablo Dávila**.
## Descripción 
Este proyecto consiste en un programa en C++ que permite registrar, almacenar y procesar notas de estudiantes mediante un sistema en consola.

El sistema fue desarrollado como parte del mini-proyecto del bootcamp "Programación de Cero a POO", aplicando estructuras básicas de programación como arreglos, funciones, validaciones y control de flujo.

## Objetivo del Respositorio
Este respositorio tiene como finalidad organizar y almacenar el material, ejercicios, practicas y proyectos desarollados durante el bootcamp, ademas de ser como evidencia del proceso de aprendizaje mediante el uso de Git y GitHub.

## Funcionalidades
- Registrar múltiples notas
- Mostrar todas las notas ingresadas
Calcular:
- Promedio general
- Nota mayor
- Nota menor
- Contar estudiantes:
- Aprobados
- Reprobados
Buscar una nota específica
